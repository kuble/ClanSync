# 밸런스메이커 UI · 기획 메모

목업(`mockup/pages/main-clan.html`)의 **맵·영웅 밴(편집 단계)** 안내는 **이 문서에만** 둔다. 해당 구간에는 `?` 툴팁을 두지 않는다. 그 외 화면(프리미엄 줄, 참고 패널, 단계 탭 등)의 짧은 안내는 목업 `data-tip`과 아래 절을 맞춘다.

PRD 상세는 `PRD.md` §6.1 등과 함께 본다.

---

## 페이지 개요

- 운영진이 **출전 명단·포지션**을 잡는 편집기.
- **5v5 · 탱1 딜2 힐2** 고정.
- **맵**은 맵 밴이 꺼져 있을 때만 직접 선택(모달). 맵 밴이 켜지면 최종 맵은 **② 맵 밴픽 세션**에서 결정(아래 스펙).
- 상단 **단계 탭**(편집 / 밴픽 / 5vs5 / 관전·예측)으로 배치 확정 이후 화면을 전환해 미리 본다.

---

## 편집 카드 · 배치 확정

- 목업은 **시나리오 미리보기**다. 실제 구현에서는 배치 확정 시 참가/밴픽 옵션·역할에 따라 **② 밴픽 / ③ 5vs5 / ④ 관전·예측** 등으로 분기·리다이렉트할 수 있다.

---

## 맵 밴 · 영웅 밴 (편집 ① — 문서 전용 카피)

아래는 목업·구현 시 **단일 소스**로 쓴다. **영웅 밴픽**은 맵 밴픽 구현 이후 별도 정리.

### 편집 ① — 맵 밴 OFF

- **맵**은 모달에서 직접 선택.

### 편집 ① — 맵 밴 ON

- **맵 라벨·맵 선택 버튼·모드 문구**는 표시하지 않는다 (`display: none` 수준). 맵 선택 예외 경로는 *「맵 밴이 켜져 있어 이 단계에서는 맵 선택이 없습니다. ② 맵 밴픽 세션에서 맵이 정해집니다.」* (목업 `alert`).
- **배치 완료 버튼 위**에 **맵 풀 분류 드롭다운**을 둔다: 전체, 쟁탈, 호위, 밀기, 혼합, 플래시 포인트 등(서버·DB의 맵 분류와 일치).
- 편집 단계에는 **밴 슬롯 UI를 두지 않는다.**

### ② 맵 밴픽 세션 (참가자)

1. **배치 완료 시점**에 참가자에게 **디스코드 알림**이 가고, 알림의 **리디렉션 링크**로 밴픽 세션(웹)이 연다.
2. 세션 **제한 시간 15초**, 남은 시간 **실시간 표시**.
3. 앞서 선택한 **분류에 해당하는 맵 풀**에서 **3개를 무작위**로 뽑아 표시한다.
4. **투표 규칙(A안)**: 경기 **참가자**는 후보 **3개 중 1개에만 1표**. 미투표는 별도 페널티 없음 — **각 맵 득표수만** 가중치로 쓴다.
5. **15초 종료 후** 세 맵의 득표수를 **가중치**로 하여 **맵 1개를 무작위 확정**한다.
6. **동률**: 득표가 **완전 동일**한 경우(예: 3·3·3 또는 0·0·0) 가중치를 **1:1:1**로 보고 **균등 1/3**으로 확정한다.
7. **확정 연출(참고)**: 오버워치 2 맵 투표는 세 후보를 순환하는 **룰렛(roll-through)** 이 멈추며, 득표가 많은 맵에 더 오래 머무는 식으로 보인다. 웹 목업에서는 **동일한 가중 확정 결과**에 맞춰 룰렛·감속·최종 하이라이트만 재현한다(게임과 1:1 동일한 타이밍·에셋은 아님).

### 단계별 화면 (워크플로 탭) — 맵 밴 기준

| 탭 | 설명 |
|----|------|
| ② 밴픽 | 맵 밴 ON: 위 **맵 밴픽 세션** UI(목업). 영웅 밴은 추후. |
| ③ 5vs5 | 맵·영웅 밴이 **모두 OFF**일 때 참가자 **고정 라인업**(읽기 전용). 맵 밴 ON이면 맵 확정 후 동일. |
| ④ 관전 | 비출전 구성원. 승부 예측 등은 Premium 범위. |

## 집계일 (KST) — 도메인 규칙

밸런스 편집 목업에는 **집계일 입력 UI를 두지 않는다.** 아래는 API·아카이브 귀속용 규칙만 정리한다.

- **당일 정의**: 해당 일자의 **06:00**(KST)부터 **다음 날 06:00** 직전까지를 같은 집계일로 본다. 자정을 넘긴 새벽 경기도 같은 집계일에 묶인다.
- **표시 일자·아카이브 귀속**은 백엔드/API에서 처리하고, 클라이언트는 응답값만 사용한다.

## 도움말 툴팁 본문 (목업 `data-tip`과 동기)

**맵·영웅 밴(편집 행)** 은 위 「맵 밴 · 영웅 밴」절만 참고한다. 나머지 화면의 `?`는 구현 시 이 절과 맞추거나 i18n 키로 분리한다.

### Premium 기능 줄 (편집 상단)

- **팀 자동 밸런싱**: Premium 클랜 기능. 자동 점수·포지션(탱·딜·힐)을 반영해 블루·레드 진영을 자동 분배. 운영진은 결과를 수동 조정 가능.
- **OCR 닉 입력**: Premium. 내전·스크림 결과 화면 스크린샷에서 닉네임 인식으로 명단 입력 보조.
- **디스코드 시작 알림(자동)**: Premium. 배치 완료 확정 시 연동 디스코드에 시작 안내·**맵 밴 ON이면 밴픽 세션 URL** 리디렉션 포함. 별도 버튼 없음.

### 승률 예측(참고 패널)

- Premium 참고 지표. 팀 자동점수 합 기반 승률 추정(목업). 실제 배당·예측 마감은 정책 따름.

### 관전 탭 · 승부 예측

- Premium. 비출전 구성원이 블루/레드 승 예측·클랜코인. 마감·배당·파리뮤추엘 정산은 서버 정책.

### 기타

- 단계 탭·밴픽·5vs5·관전 등 나머지 `?` 본문은 목업 `data-tip`과 동기 유지한다.
