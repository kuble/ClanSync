# 권한·구독 게이팅 매트릭스

> "누가 뭘 볼 수 있고, 뭘 할 수 있나"를 한 표에 모은다. 페이지 문서들의 "권한·구독에 따른 차이" 섹션은 이 표의 발췌본이다. 변경은 여기를 우선 갱신한다.

## 0. 표 읽는 법

- **역할** (게임 길드 비유): `leader` 클랜장 / `officer` 운영진 / `member` 구성원 / `guest` 비로그인·비소속
- **구독**: `free` 기본 / `premium` 유료
- **값**:
  - ✓ 보임·사용 가능
  - ✗ 숨김 (메뉴 자체 안 보임)
  - 🔒 보이긴 하지만 잠금 (업그레이드/권한 안내 표시)
  - — 해당 없음

> 역할은 **포함 관계**가 아니라 **개별 권한**이다. "운영진+"이라고 적힌 곳은 leader/officer 모두 가능, "클랜장만"은 leader만.
