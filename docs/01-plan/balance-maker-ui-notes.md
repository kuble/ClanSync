# 밸런스메이커 UI · 기획 메모

목업(`mockup/pages/main-clan.html`)의 **맵·영웅 밴(편집 단계)** 안내는 **이 문서에만** 둔다. 해당 구간에는 `?` 툴팁을 두지 않는다. 그 외 화면(프리미엄 줄, 참고 패널, 단계 탭 등)의 짧은 안내는 목업 `data-tip`과 아래 절을 맞춘다.

PRD 상세는 `PRD.md` §6.1 등과 함께 본다.

---

## 페이지 개요

- 운영진이 **출전 명단·포지션**을 잡는 편집기.
- **5v5 · 탱1 딜2 힐2** 고정.
- **맵**은 맵 밴이 꺼져 있을 때만 직접 선택(모달). 맵 밴이 켜지면 최종 맵은 **② 맵·영웅 밴픽 세션**에서 결정.
- 상단 **단계 탭**(편집 / 밴픽 / 5vs5 / 관전·예측)으로 배치 확정 이후 화면을 전환해 미리 본다.

---

## 편집 카드 · 배치 확정

- 목업은 **시나리오 미리보기**다. 실제 구현에서는 배치 확정 시 참가/밴픽 옵션·역할에 따라 **② 밴픽 / ③ 5vs5 / ④ 관전·예측** 등으로 분기·리다이렉트할 수 있다.

---

## 맵 밴 · 영웅 밴 (편집 ① — 문서 전용 카피)

아래는 과거 목업 `?`에 두었던 설명을 통합·보존한 것이다. **구현·i18n 시 이 문단을 단일 소스로 쓴다.**

### 동작 요약

- 편집 단계(①)에서는 **밴 슬롯 UI를 두지 않는다.** 맵·영웅 밴 슬롯은 **② 맵·영웅 밴픽 세션**에서만 사용한다. 슬롯 개수·적용 규칙은 추후 확정.
- **맵 밴 ON**인 경우: 편집 단계에서 **맵 라벨·맵 선택 버튼·모드 문구(예: `· 제어`) 전체를 표시하지 않는다** (`display: none` 수준, 비활성만이 아님). 최종 맵은 ② 밴픽 세션에서 정한다.
- 맵 밴이 켜진 상태에서 맵 선택을 호출하는 예외 경로(키보드·스크립트 등)가 있으면: *「맵 밴이 켜져 있어 이 단계에서는 맵 선택이 없습니다. ② 맵·영웅 밴픽 세션에서 맵이 정해집니다.」* 를 안내한다(목업은 `alert`).

### 맵·영웅 밴 옵션 일반 안내 (구 목업 상단 `?` 본문)

- 맵 밴·영웅 밴을 켜면 **② 맵·영웅 밴픽 세션**에서 슬롯을 사용한다. 편집 단계에는 슬롯을 두지 않는다. 슬롯 개수·규칙은 추후 확정된다.

### 맵 밴 ON 시 맵 UI 숨김 (구 목업 맵 밴 스위치 옆 `?` 본문)

- 맵 밴이 켜져 있어 **편집 단계에서는 맵 선택 UI가 보이지 않는다.** 최종 맵은 **② 맵·영웅 밴픽 세션**에서 정한다.

## 집계일 (KST) — 도메인 규칙

밸런스 편집 목업에는 **집계일 입력 UI를 두지 않는다.** 아래는 API·아카이브 귀속용 규칙만 정리한다.

- **당일 정의**: 해당 일자의 **06:00**(KST)부터 **다음 날 06:00** 직전까지를 같은 집계일로 본다. 자정을 넘긴 새벽 경기도 같은 집계일에 묶인다.
- **표시 일자·아카이브 귀속**은 백엔드/API에서 처리하고, 클라이언트는 응답값만 사용한다.

## 단계별 화면 (워크플로 탭)

| 탭 | 설명 |
|----|------|
| ② 밴픽 | 맵 또는 영웅 밴픽이 켜진 경기에서 **참가자**에게 보이는 단계(진영별 번갈아 선택 등 목업). |
| ③ 5vs5 | 맵·영웅 밴이 **모두 OFF**일 때 참가자에게 보이는 **고정 라인업**(읽기 전용). |
| ④ 관전 | 이번 경기에 **출전하지 않은** 구성원 화면. 승부 예측·배당·파리뮤추엘 등은 **Premium** 범위. |

---

## 목업 허브 · 플랜

- `_hub.html`에서 **Free / Premium** 전환 시 Premium 전용 UI(예: 승부 예측 블록) 노출이 달라진다.
- Free 플랜 목업에서 해당 UI는 숨기고, **도움말**로 Premium에서 확인하도록 안내한다.

## 도움말 툴팁 본문 (목업 `data-tip`과 동기)

**맵·영웅 밴(편집 행)** 은 위 「맵 밴 · 영웅 밴」절만 참고한다. 나머지 화면의 `?`는 구현 시 이 절과 맞추거나 i18n 키로 분리한다.

### Premium 기능 줄 (편집 상단)

- **팀 자동 밸런싱**: Premium 클랜 기능. 자동 점수·포지션(탱·딜·힐)을 반영해 블루·레드 진영을 자동 분배. 운영진은 결과를 수동 조정 가능.
- **OCR 닉 입력**: Premium. 내전·스크림 결과 화면 스크린샷에서 닉네임 인식으로 명단 입력 보조.
- **디스코드 시작 알림(자동)**: Premium. 배치 완료 확정 시 연동 디스코드 채널에 시작 안내 자동 발송. 별도 버튼 없음.

### 승률 예측(참고 패널)

- Premium 참고 지표. 팀 자동점수 합 기반 승률 추정(목업). 실제 배당·예측 마감은 정책 따름.

### 관전 탭 · 승부 예측

- Premium. 비출전 구성원이 블루/레드 승 예측·클랜코인. 마감·배당·파리뮤추엘 정산은 서버 정책.

### 기타

- 단계 탭·밴픽·5vs5·관전 등 나머지 `?` 본문은 목업 `data-tip`과 동기 유지한다.
