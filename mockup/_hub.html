디버그 · 밸런스 세션 플로우

A 밸런스장(세션 연 운영진 1명) · B 그 외 운영진·구성원(동일 관전 동기화). 목업에서는 클랜장·운영진 모두 편집 UI — 실제 구현 시 A만 ① 편집.

  1. 세션 생성 (A가 밸런스 메이커 시작)
  2. ① 배치 — A 편집 / B 실시간 경기 보드
  3. 배치 완료 → 밴 ON이면 ② 밴픽 화면으로 리디렉션
  4. ② 밴픽 — 참가자 투표 · 비참가 B는 실시간 관전
  5. 경기 시작 = 밴 종료 직후 (밴 없으면 배치 완료 직후)
  6. ③ 경기 진행 — 예측 마감 타이머 시작 · 결과 입력
  7. 동점·무효·재경기 → 해당 판 예측 무효
  8. 사후 정정 → 클랜 관리 · 내전 히스토리(예정)
단계 A 밸런스장 B (타 운영진·구성원)
① 배치 편집 UI 경기 진행 동기(읽기)
② 밴픽 밴픽 UI(참가 시) 참가: 밴픅 / 비참가: 실시간 관전
③ 경기·예측 왕관·결과 입력 비참가: 승부예측(Premium)
